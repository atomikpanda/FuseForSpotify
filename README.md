## Fuse for Spotify

### Installation:

1. Open the .xcworkspace file with Xcode
2. Ensure breakpoints are disabled under Debug->Deactivate Breakpoints in Xcode
3. Run the Xcode project on an iOS device or by using the iOS simulator.
